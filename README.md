# RPSLS Online

A browser-based Rock, Paper, Scissors, Lizard, Spock game built as part of Launch School curriculum work for PY175.

## Overview

This project implements the classic RPSLS game with an online web based interface, allowing a player to play against a computer opponent.

## Features

- Play Rock, Paper, Scissors, Lizard, Spock
- Optional Sign-in feature with leaderboard for players with accounts
- Simple, accessible web interface
- Built for learning and demonstration purposes

## Getting Started

1. Clone the repository.
2. Open the application in your browser.

## Notes

This project is part of Launch School coursework (PY175) and is intended to demonstrate understanding of game logic and web app structure.
